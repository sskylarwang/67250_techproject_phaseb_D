Current Exhibitions

Digital Dreams: Art in the Age of Algorithms

This exhibition explores how artists use algorithms, artificial intelligence, and generative tools to create new visual forms. Visitors can experience interactive displays and see how machines and human creativity work together in real time.

Code & Canvas: The Language of Creative Technology

Code is increasingly becoming a medium for artistic creation. Featuring works by digital artists and designers, this exhibition highlights installations, projections, and interactive pieces that reveal how lines of code can become immersive visual experiences.

Signal / Noise

Signal / Noise examines how modern life is shaped by data streams, notifications, and constant connectivity. The exhibition invites visitors to reflect on how technology affects attention, memory, and communication.
